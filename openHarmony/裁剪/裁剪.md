
组件级裁剪
```
通过修改build目录下的配置文件：
//build/config/component/xxx.json
{
    "components":[
        {
          "component": "ace_engine",
          "features":[]
         },
         {
           "component": "account_os_account",
           "features":[]
         }
         //可添加或删除组件
    ]
}

```
功能模块裁剪，修改kernel配置：
```
//使用menuconfig 工具配置Linux 内核
make menuconfig
```
