
指针
```

指针是一个存储内存地址的变量（本质是一个变量），指向其他变量或者对象。

int *p; //创建一个可以存储int类型地址的变量。


指针有类型。它指向特定类型的数据，如（int*  char*）

作用：
1、直接访问内存：允许直接访问和操作内存地址，
这对高效内存管理和性能优化非常重要。

2、传递参数：使用指针作为函数参数，可以避免复制大数据结构，提高效率。
可以直接修改传入对象的值。

3、实现数据结构：指针是实现链表、树、图等复杂数据结构的基础，允许灵活的管理和连接节点。

4、通过指针操作数组元素。

注意点：
想要操作指针本身（让它指向哪里）： 用 p = ...
想要操作指针指向的值：用 *p = ...


通过*a 可以获取a的值。
通过&a 可以获取a的内存地址（结果是一个常量）
```

使用指针避免内存泄漏的方案
```
一、
int* ptr = new int(10);
//使用 ptr

delete ptr; //释放内存

二、
c++11 使用 std::unique_ptr 和 std::shared_ptr 自动管理内存

std::unique_ptr<int> ptr(new int(10)); //自动释放

三、避免悬空指针
在删除指针后，将其设置为nullptr，防止后续代码中误用

delete ptr;
ptr = nullptr;
```

```
Node* next；//声明一个指针变量
这个next 包含了地址，这也就是链表插入方便的原因，只要改变next，赋值其他的指针，就相当于换了个地址。也就指向其他地方了。


void swap_correct(int *a, int *b){ //参数 int *a 是指针变量，这个的意思是接收地址值 （传递x，y的地址）也就是指针值
    int temp = *a; //获取a所指向地址的值
    *a = *b;
    *b = temp;
}

int main(){
    int x = 10, y = 20;
    swap_correct(&x, &y);
    printf("%d, %d\n", x, y); //输出：20，10  （成功交换）
    return 0;
}



将b的地址给到a
void  test(int* a, int* b){
	a = b;  //将b的地址给到a, 那么地址中的值也就变成20
	printf("a的值：%d", *a);  //输出20
}

int main()
{
	int x=10, y=20;
	test(&x, &y);
    return 0;
}
```
