
宏定义
```
可以用来复用代码

作用：在编译时进行文本替换，替换其对应的值。
提高代码的可读性和可维护性，避免重复代码




#include <iostream>
#define SIZE 25

int main() {
    int arr[SIZE];  // 定义一个大小为 25 的数组

    // 初始化数组
    for (int i = 0; i < SIZE; i++) {
        arr[i] = i * 2; // 将数组元素设置为 0, 2, 4, ..., 48
    }

    // 输出数组
    for (int i = 0; i < SIZE; i++) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}
```
