
```
applications  应用程序demo
kernel        内核子系统，负责任务调度，内存管理等常见内核功能
hiviewdfx     可维可测子系统，提供日志相关功能
communication 通信子系统，包括wifi，蓝牙功能
iothardware   IOT 外设系统。提供常见的外设接口。
startup       启动子系统。内核启动后运行的第一个子系统，负责 在内核启动后 到应用启动前 系统关键进程和服务的启动过程的功能
update        升级子系统。用来支持Openharmony 设备的OTA 升级  （Over-The—Air）
utils         公共基础库子系统，提供了一些常见的c、c++开发增强api
distributed_schedule 分布式调度子系统。负责跨设备部件管理，提供访问和控制远程组件的能力，支持分布式场景下的应用协同
security      安全子系统。包括系统安全，数据安全，应用安全等
test          测试子系统。
```
